## 📊 EXEMPLES D'UTILISATION IMMÉDIATE

### 👤 Créer un Patient avec Numéro Unique
```bash
curl -X POST http://localhost:8080/api/patients \
  -H "Content-Type: application/json" \
  -d '{
    "firstName": "Marie",
    "lastName": "Dubois", 
    "dateOfBirth": "1985-03-15",
    "phone": "0123456789"
  }'
# Retourne: {"patientNumber": "P1703612345abc12345", ...}
```

### 📊 Ajouter à la File d'Attente avec Priorité IA
```bash
curl -X POST "http://localhost:8080/api/queue/add?patientId=1&departmentId=1" \
# Retourne: {"queueNumber": "URG001", "priority": 4, "estimatedWaitTime": 15}
```

### 💊 Dispenser une Prescription avec Vérification Stock
```bash
curl -X PATCH "http://localhost:8080/api/pharmacy/prescriptions/1/dispense?pharmacistId=2"
# Vérifie stock + interactions + réduit automatiquement le stock
```
